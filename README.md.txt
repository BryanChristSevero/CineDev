## Descrição do Projeto

Este projeto é um protótipo de sistema em Java para gerenciamento de assentos e compra de bilhetes da empresa CineDev. O sistema permite a seleção de assentos disponíveis e a compra de ingressos de forma simples e funcional.

## Como compilar e executar

1. Abra o terminal na raiz do projeto.
   
2. Para compilar, digite:
   javac CineDev.java

3. Para executar o programa, use o comando:
   java CineDev
